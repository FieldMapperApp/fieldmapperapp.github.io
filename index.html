About

FieldMapper is a fieldwork toolbox for the collection of geo-referenced data. The app (available for Android tablets only) is free and lightweight. Just take a set of variables and customise FieldMapper's map interface according to your needs. Start surveying bounded systems or interviewing people now—because it matters where things happen!

Learn more about how FieldMapper works in the Documentation. FieldMapper is open-source—contributions are welcome!

Download

Download the APK here. Open the linked file on your tablet and install the app. You need to allow installation from unknown sources in the settings if you haven't.
